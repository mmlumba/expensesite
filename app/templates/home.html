Best Practices:

  • The recommended web browsers are: Chrome or Safari on iPad, Chrome or Internet Explorer for Windows and Chrome for Mac.
  • Web browsers must be set to "Always Accept Cookies"
  • Collect receipts before you begin a new expense report
  • Save progress frequently and always save progress after uploading your receipts
  • Check the "I accept" box on the last page to "sign" your expense report
  • Save Progress on the last page of the web form
  • Use the SUBMIT button to send your expense report to NRECA
  • View the confirmation page and save the confirmation email from Bernita
